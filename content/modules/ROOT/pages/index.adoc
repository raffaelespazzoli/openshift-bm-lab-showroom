= OpenShift Bare Metal Deployment

Welcome to the OpenShift Bare Metal Deployment workshop!

== What you'll learn

In this workshop, you will:

* Understand bare metal infrastructure requirements for OpenShift
* Configure network settings for bare metal clusters
* Deploy and verify a production-ready OpenShift cluster on bare metal
* Troubleshoot common bare metal installation issues

== Who this is for

This workshop is designed for Platform Engineers, Red Hat Consultants, and Red Hat Specialists who want to practice the installation of OpenShift bare metal clusters.

**Experience level**: Intermediate

== Prerequisites

Before starting this workshop, you should have:

* Intermediate knowledge of OpenShift and container platforms
* Familiarity with Linux command line operations
* Understanding of networking concepts (VLANs, DNS, DHCP)
* Access to the lab environment provided

== Workshop environment

This workshop provides a complete bare metal lab environment with pre-configured infrastructure including:

* Bastion host for installation and management
* Multiple bare metal nodes (virtualized) ready for cluster deployment
* Network services: DNS, NTP, proxy, and storage
* Redfish BMC simulation for hardware management

== Estimated time

This workshop will take approximately **3 hours** to complete.

You can work at your own pace through the 3 modules:

* Module 1: Infrastructure Preparation (45 minutes)
* Module 2: OpenShift Installation (2 hours)
* Module 3: Post-Installation and Verification (30 minutes)
